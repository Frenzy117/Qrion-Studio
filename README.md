# 🧠 Qrion — The Ultimate Prompt Engineering Playground

**Qrion** is a sleek, developer-focused web tool designed to help you craft, test, and fine-tune prompts across multiple AI models — all in one place. It features structured prompt input, system instruction support, and conversation context control, and various parameters to help you get the most out of today’s powerful LLMs.

---

## ✨ Features

- 🔧 **System Instruction Support**  
  Tailor model behavior with dedicated system instructions (Gemini, LLaMA via Groq, Mistral via smart prepending).

- 🧩 **Conversational Context Handling**  
  Prepend prior context to your prompt for coherent multi-turn interactions.

- 🧠 **Prompt Engineering Workspace**  
  Write, refine, and test prompts in a structured environment with live feedback.

- 🤖 **Multi-Model Support**  
  Test prompts across **Gemini**, **LLaMA (Groq)**, **Mistral**, and more (like DeepSeek coming soon).

- 💾 **Prompt Library Integration**  
  Paste and reuse high-performing prompts directly into the chat box.

- 🌟 **Beautiful UI**  
  Crafted with Tailwind CSS & Sparkles UI (Aceternity) for a clean, modern look.

---

## 🧰 Tech Stack

- **Frontend:** React.js, Tailwind CSS, Aceternity UI, Material UI
- **Backend:** Java (Spring Boot MVC), REST APIs  
- **Database:** PostgreSQL  
- **AI APIs:** Gemini (Google), Groq (LLaMA, Mistral), more coming soon  
- **Hosting/Infra:** Google Cloud Platform (Firebase, Cloud Run)

## 🗣️ Why Qrion?

Whether you're testing model behavior, writing educational prompts, or experimenting with prompt styles, Qrion gives you **fine-grained control** over every part of your interaction — from the system’s tone to the conversational context.
